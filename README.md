## Pasos

Clonar repositorio

**`git clone <repository> --branch <dev> <directory>`**

Instalar las dependencias necesarias

**`npm i`**

Ejecutar **[App](http://localhost:3000)** en modo de desarrollo

*`npm start`**

Crear **`build`** de producción

**`npm run build`**
