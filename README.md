## Pasos

Clonar repositorio

		git clone <repository>

Instalar las dependencias necesarias

		npm i

Ejecutar app en modo de desarrollo

		npm start

Crear **build** de producción

		npm run build

## Elementos necesarios

**Variables de entorno en un archivo `.env`**

		# Url de la API
		REACT_APP_BACKEND_URL=""
